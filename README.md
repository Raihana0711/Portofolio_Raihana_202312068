## 🚀 Cara Menjalankan Proyek

### Prerequisites
- Web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code, Sublime Text, Atom)
- Web server lokal (XAMPP, WAMP, MAMP) untuk file PHP

### Menjalankan File HTML/CSS/JavaScript
1. Clone repository ini:
   ```bash
   git clone https://github.com/Raihana0711/Portofolio_Raihana_202312068.git
   ```
2. Navigasi ke folder modul yang diinginkan
3. Buka file `.html` di web browser

### Menjalankan File PHP
1. Pastikan web server lokal (XAMPP/WAMP) sudah terinstall
2. Pindahkan folder proyek ke direktori `htdocs` (XAMPP) atau `www` (WAMP)
3. Jalankan Apache server
4. Akses melalui browser: `http://localhost/Portofolio_Raihana_202312068/`
